## Interpretation

This is safety-positive evidence that the logit-safe seed-only effect survives scaling from Qwen2.5-0.5B to Qwen2.5-1.5B, at least for early layers under the current fixed seed schedule.

It is not yet runtime-positive evidence for Qwen2.5-1.5B at B4. The next gate is a direct seed-only runtime benchmark on L0 and L3 against dense exact for the same Qwen2.5-1.5B 32K/B4 shape.

Decision:

```text
if Qwen1.5B L0/L3 B4 runtime wins:
  package Qwen1.5B B4 policies

elif Qwen1.5B L0/L3 safety stays green but runtime loses:
  build or enable head-private split-seed for B4

else:
  run seed schedule search before expanding Qwen1.5B policies
```
